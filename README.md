# Physics Experiment Recorder

Live site: https://baditaflorin.github.io/physics-experiment-recorder/

Repository: https://github.com/baditaflorin/physics-experiment-recorder

Physics Experiment Recorder turns phone video of tagged motion experiments into
position-vs-time data, curve fits, and reproducible exports in a static browser app.

## Quickstart

```sh
npm install
npm run dev
npm run build
npm test
```

## Deployment

This is a Mode A GitHub Pages app. The Vite build writes the publishable site to
`docs/`, and GitHub Pages serves `main` branch `/docs`.
