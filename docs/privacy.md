# Privacy

The app does not include analytics, trackers, accounts, or a backend.

Videos, CSV files, generated tracks, fits, corrections, and exports stay in the
browser unless the user explicitly downloads or shares them. IndexedDB stores the
latest local experiment record on the same device and browser profile.

The page links to:

https://github.com/baditaflorin/physics-experiment-recorder

https://www.paypal.com/paypalme/florinbadita

Those external sites have their own privacy policies.
