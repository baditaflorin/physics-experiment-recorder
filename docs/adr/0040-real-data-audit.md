# 0040 - Real-Data Audit Findings

## Status

Accepted

## Context

Phase 2 Substance uses ten messy fixture tracks as the grading rubric.

## Decision

Prioritize unit inference, headerless CSV inference, incomplete-row recovery,
outlier surfacing, deterministic fixture tests, and actionable domain errors.

## Consequences

The app becomes more useful for ordinary classroom exports without changing the
Mode A architecture or adding new product surface.

## Alternatives Considered

Polish work and new features were rejected until the fixture pass rate improves.
